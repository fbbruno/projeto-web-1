Banco de Dados II (BD2A4) – 66,7h

Estudo de linguagem de descrição e manipulação de dados. Estudo de regras de integridade e dependências funcionais. Análise das formas de normalização. Análise sobre gerenciamento de transações.

Banco de Dados

Metodologia de Pesquisa (MTPA4) – 33,3h

Estudo da metodologia científica. Análise do método científico. Elaboração de artigo científico. Análise da elaboração de textos

Metodologia de Pesquisa

Gestão de Serviços Informatizados (GSIA4) – 33,3h

Gestão de Serviços Informatizados

Análise da implantação de um servidor. Estudo da implantação dos diferentes tipos de serviços de um servidor.

Implantação de Servidores

Análise do gerenciamento de projetos. Estudo das etapas do projeto de um sistema. Elaboração de um projeto de software.

Projetos de Sistemas

Estudo de uma linguagem orientada a objetos. Desenvolvimento de sistemas com interface gráfica utilizando programação orientado a objetos e conexão com banco de dados.
